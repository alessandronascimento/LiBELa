Detailed Description

The PRMTOP class has a number of methods dedicated to parse a AMBER prmtop file and get all the necessary information for energy evaluation in a receptor-ligand system. There is also a method to parse the atomic coordinates of a inpcrd file. The parser works well for ligand and receptor prmtop/inpcrd files.
