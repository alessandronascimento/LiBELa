void TEMP_SCHEME::sa_run (  ) 

The sa_run method defines a simulated annealing run with initial temperature defined by the user. The user also defines how many steps will be simulated before the temperature is decreased. The temperature decrease is not linear in McLibela. Rather than linearly, the temperature is decreased by 10*log(t) in each SA step.

rmsd_energy is defined as an output file to keep the trial number, the rmsd and the evaluated energy.
